論理回路シミュレーションは 純粋 Haskell で実装。
GHC WASM バックエンドでコンパイルしてブラウザで動作します。

ビルド方法:

# GHC WASM のインストール
make install-wasm-ghc
source ~/.ghc-wasm/env

# WASM ビルド
make wasm

# サーバー起動
make serve   # → http://localhost:8080
